Rock Paper Scissor

This Rock Paper Scissors game is a Flask-based web application where players compete against the computer by selecting Rock, Paper, or Scissors. The computer randomly picks its choice, and the winner is determined based on traditional game rules. Each round's results, including the player's choice, computer's choice, and outcome, are stored in a CSV file for future analysis. The game also provides statistical insights, such as the total number of times each choice was made, win/loss/draw counts, and percentage calculations. Flask handles the backend logic, while HTML and CSS are used for the frontend.

CPU Scheduler System

This project is a CPU Scheduling System built using Flask, which simulates different scheduling algorithms including Non-Preemptive, Preemptive, Round Robin, and Priority Non-Preemptive Scheduling. The system takes process details as input, applies the selected scheduling algorithm, and visualizes the execution order, turnaround time, and waiting time. It helps in understanding and comparing CPU scheduling techniques by displaying real-time results through a web-based interface. This project improved my skills in Flask, algorithm implementation, and data visualization.
